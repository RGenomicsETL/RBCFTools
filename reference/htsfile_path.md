# Get Path to htsfile Executable

Returns the path to the bundled htsfile executable for identifying file
formats.

## Usage

``` r
htsfile_path()
```

## Value

A character string containing the path to the htsfile executable.

## Examples

``` r
htsfile_path()
#> [1] "/home/runner/work/_temp/Library/RBCFTools/htslib/bin/htsfile"
```
