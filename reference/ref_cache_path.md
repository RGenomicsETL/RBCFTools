# Get Path to ref-cache Executable

Returns the path to the bundled ref-cache executable for reference
sequence cache management.

## Usage

``` r
ref_cache_path()
```

## Value

A character string containing the path to the ref-cache executable.

## Examples

``` r
ref_cache_path()
#> [1] "/home/runner/work/_temp/Library/RBCFTools/htslib/bin/ref-cache"
```
