# Get Path to annot-tsv Executable

Returns the path to the bundled annot-tsv executable.

## Usage

``` r
annot_tsv_path()
```

## Value

A character string containing the path to the annot-tsv executable.

## Examples

``` r
annot_tsv_path()
#> [1] "/home/runner/work/_temp/Library/RBCFTools/htslib/bin/annot-tsv"
```
