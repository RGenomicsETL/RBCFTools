# Get Path to bcftools Plugins Directory

Returns the path to the directory containing bcftools plugins.

## Usage

``` r
bcftools_plugins_dir()
```

## Value

A character string containing the path to the bcftools plugins
directory.

## Examples

``` r
bcftools_plugins_dir()
#> [1] "/home/runner/work/_temp/Library/RBCFTools/bcftools/libexec/bcftools"
```
