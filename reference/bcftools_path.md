# Get Path to bcftools Executable

Returns the path to the bundled bcftools executable.

## Usage

``` r
bcftools_path()
```

## Value

A character string containing the path to the bcftools executable.

## Examples

``` r
bcftools_path()
#> [1] "/home/runner/work/_temp/Library/RBCFTools/bcftools/bin/bcftools"
```
